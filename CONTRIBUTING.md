1. **Fork and Clone:** Fork the repository and clone it to your local machine.
   ```sh
   git clone https://github.com/YOUR_USERNAME/Nepal-College-Email-Harvester.git
   cd Nepal-College-Email-Harvester
   ```

2. **Install Dependencies:**
   ```sh
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file and add your Gemini API key.
   ```
   VITE_GEMINI_API_KEY=YOUR_API_KEY
   ```

4. **Run the Development Server:**
   ```sh
   npm run dev
   ```

## Pull Request Guidelines

- **Title:** Your PR title should be clear and descriptive.
- **Description:** Provide a detailed description of the changes you've made.
- **Link to Issue:** If your PR addresses an open issue, please link to it.
- **Screenshots:** If your changes affect the UI, please include screenshots.
